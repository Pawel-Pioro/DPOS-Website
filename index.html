Welcome to DPOS


What is DPOS?

DPOS was created by 2 programmers in the python language. They worked on it to make something great. It works as an application an top of your existing OS.

DPOS can:

  • Check your internet connection
  • Store files in folders through its file manager that can be changed through the source code
  • Store and use applications such as Google, Word or Python IDLE
  • Play a selection of games that are preinstalled
